Weekday OT rules

Only one block size applies at a time — the government default is 15 minutes. Tick 30 or 45 instead only if this institution uses a larger block.

If someone arrives late (within the grace period), they must cover that time after the standard end time before OT starts counting.

The first full hour of overtime must be completed before any OT is granted — after that, time is counted in the block size selected above. This applies to weekdays, weekends and holidays alike.

Weekend / holiday rules

On weekends and public holidays, the entire time worked counts as overtime, calculated in the same block size from the actual time in to time out.
